## Pre-Installation

```sh
$ sudo apt install gcc-multilib g++-multilib
$ sudo apt install doxygen graphviz texlive-base
```

## Testing

```sh
$ make
$ make run
```

## Documentation

```sh
$ doxygen -g
$ doxygen
```
