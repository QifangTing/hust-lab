# Tao Lang Compiler

## Build

### With Makefile

```sh
$ make build
$ make run
```

### With Gradle

*   Install Gradle on Linux

```sh
$ sudo apt install gradle
```

*   Get Help Informations

```sh
$ gradle help
$ gradle tasks
```

*   Basic Tasks

```sh
$ gradle assemble
$ gradle build
$ gradle clean
$ gradle test
$ gradle jar
$ gradle javadoc
```

## Run/Test

```sh
$ make run
$ make test
```
