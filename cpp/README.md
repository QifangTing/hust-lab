## Pre-Installation

```sh
# install GNU c collection and g++ compiler
$ sudo apt install build-essentials gcc-multilib g++-multilib

# install doxygen(documentation generator)
$ sudo apt install doxygen graphviz
```

## Run

```sh
$ make clean
$ make
$ make run
```

## Debug

```sh
$ make debug
```
