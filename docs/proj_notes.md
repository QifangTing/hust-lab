# Hust OS Final Project Notes

## Proj 1-2 : Multi-Process Electron Windows (Based on V8 JS Engine and Nodejs)

### Set Up

```sh
$ sudo apt install nodejs npm
$ sudo npm install -g cnpm
```

### Build and Run

```sh
$ cnpm install
$ cnpm start
```

### Environment

*   Ubutnu 16.04
*   Atom Editor v1.13.0
*   Node v6.6.0
*   Npm v3.10.3
*   Electron v1.4.13
