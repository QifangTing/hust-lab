## Installtion

### Linux

```sh
$ sudo apt-get install nodejs npm
$ sudo npm install -g cnpm
$ cnpm install
```

### Windows

*   请自行到 nodejs.org 下载并安装 nodejs v6.6.0 LTS 环境
*   一般情况下, npm 包管理器会随着 nodejs 一并安装, 若未成功, 则请自行到 https://github.com/npm/npm/releases 下载安装

```sh
$ node -v
$ npm -v
$ npm install -g cnpm
$ cnpm install
```

## Run

```sh
$ npm start
```

## Test

```sh
$ npm test
```
