# Basic notes

## Set up

```shell
git clone https://github.com/nju-ics/ics2015
sudo apt-get install libreadline-dev build-essential gcc-doc gdb git time
```

```shell
config/Makefile.git
STU_ID=141220000            # your student ID
make
```

## Basic Tools

```shell
make  clean
make gdb
make cpp
make clean-cpp
make run
make submit
```
